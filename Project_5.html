Herbalife B2B



In Mexico the Herbalife nutrition clubs are very popular, they're small places where people go to buy multiple products, every month low incomes, middle incomes or high incomes gets to the store depending on sales that were made is obtained, I developed a mobile application in android called B2B (Best Two Best) for the administration of its products, sales and customers, consisting of three types users: Host users which were to be the root user, the fellows who were the workers and Partners who were the customers. The mobile app was developed for Android Jelly Bean+, taking the whole design of the company.


Responsibilities:

  • Performed of a database for an enterprise level using phpMyAdmin.
  • Used php files to connect to the database.
  • Performed all the mobile design of the system using PSD and AI to design the icons, styles files and xml files.
  • Performed all documentation required by my client using the methodology of Scrum.
  • Used javamail to send all the mails from user registration.
  • Used Android Studio as tool to develop the mobile application.
  • Used a URL Webservice to recibe the JSON atributes.
  • Used a java class as a security login.

Environment: Java 7, Android Studio, javaMail, phpMyAdmin, php, MySQL, Illustrator, Photoshop and masOS El Capitan.

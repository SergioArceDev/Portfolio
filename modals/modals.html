SIIEM

The Department of Education Morelos Mexico for many procedures daily basis such as: the arrival of corresponding department state correspondence and sending correspondence by the department, handling all profits and income that are offered to middle school, high schools and public universities in the state, the telephone directory which includes school principals, mayors and state representatives. It is responsible for organizing what is a collegial body which sends its quarterly binnacle for review. In all that procedures already mentioned paper it was used but with the advent of the system development called SIIEM everything became agile and modern for the department of education.


Responsibilities:

  • Performed of a database for an enterprise level using MySQL Workbench.
  • Creating Users with permissions and security restrictions using Struts 2 controller and XML.
  • Performed the backend of the system using an MVC environment(BEANS, DAOS and the Action classes).
  • Performed all the user design of the system using PSD and AI to design the icons, HTML5, CSS3, Bootstrap and Jquery for all the frontend.
  • Performed all documentation required by my client using the methodology of Scrum.
  • Performed ajax functions to push all the Struts2 actions using JSON.
  • Worked on Jquery form validations for all types of input fields across the application.
  • Used javamail to send all the mails from user registration to permit.
  • Used Netbeans as tool to develop the application.
  • Performed all the design of the reports using Jasper Report.
  • Responsive design using bootstrap.

Environment: Java 7, Struts2, javaMail, Ajax, Jquery, Bootstrap, HTML5, CSS3, MySQL, Glassfish, Illustrator, Photoshop and masOS El Capitan.
